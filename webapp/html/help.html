TOMI Help
  • User Role
  • admin: Reserved and not open yet.
  • manager: Manager is the highest manager and has full permission in TOMI. He can control operator's action like lock some records and operator can't change them.
  • operator: Normal user in TOMI who can input/manage/export incident records and check reports.
  • guest: Has same priviledge with operator role except modification, he only have view permission in TOMI.
  • upgrader: A role use for map upgrading purpose, only has one difference with operator role: Not change "Modifier" when update a record.
